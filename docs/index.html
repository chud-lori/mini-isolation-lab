Mini Isolation Lab

This is a step-by-step course for engineers who want to understand containerization and Firecracker-style microVMs from first principles. The course starts with syscalls and memory, then builds toward namespaces, cgroups, a small container runtime, KVM, and a tiny VMM event loop.

How To Use This Course

Read the parts in order. Each lesson has its own page, source file, run commands, expected output, and checkpoints. The early parts are intentionally small because they introduce the kernel ideas used by the later runtimes.

The container side teaches how Linux isolates host processes with namespaces, root filesystems, and cgroups. The microVM side teaches how a host process uses KVM to run guest code behind a stronger VM boundary.

Run The Code

The demos are Linux-only. macOS does not expose Linux namespaces, cgroups, or KVM.

sudo apt-get install build-essential busybox-static
make
make parts
